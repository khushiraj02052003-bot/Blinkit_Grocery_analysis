# Blinkit_Grocery_analysis
📊 BlinkIT Grocery Data Analysis – End-to-End Project
📌 Overview

This project presents a complete end-to-end data analytics workflow using the BlinkIT grocery dataset. The goal is to transform raw retail data into meaningful business insights by applying data cleaning, exploratory data analysis (EDA), SQL queries, and interactive dashboarding. The project highlights how data-driven decisions can improve retail performance.

📁 Dataset

The dataset contains 8,523 records and 12 features, including:

Product Details: Item Type, Fat Content, Weight, Visibility
Outlet Information: Outlet Type, Size, Location Tier, Establishment Year
Performance Metrics: Sales and Ratings

This dataset is used to analyze product performance, outlet efficiency, and overall sales trends.

🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
SQL (PostgreSQL / MySQL / SQL Server)
Power BI (Dashboard & Visualization)
Excel (Initial data handling)
Gamma (Presentation creation)
🔄 Project Steps
Data Loading
Imported BlinkIT dataset using Python (Pandas)
Data Cleaning
Handled missing values (e.g., Item Weight)
Standardized categorical values (e.g., Fat Content)
Removed duplicates and corrected inconsistencies
Exploratory Data Analysis (EDA)
Analyzed sales distribution across item types and outlet types
Studied the impact of outlet size and location tier
Visualized trends using charts and graphs
SQL Analysis
Wrote queries to calculate KPIs (Total Sales, Avg Sales)
Performed category-wise and outlet-wise analysis
Derived percentage contributions and rankings
Data Visualization
Built an interactive Power BI dashboard with filters and KPIs
Reporting & Presentation
Created a detailed report summarizing insights
Designed a presentation using Gamma
📊 Dashboard

The Power BI dashboard includes:

📈 Total Sales and Average Sales KPIs
🛒 Sales by Item Type and Fat Content
🏬 Outlet-wise performance comparison
🌍 Sales by Location Tier (Tier 1, 2, 3)
🎯 Interactive filters for deeper insights
📈 Results & Insights
Supermarket-type outlets generate higher sales compared to smaller stores
Tier 3 locations contribute significantly to total revenue
Low Fat and Regular products show different sales patterns
Item visibility and type influence overall sales performance
🚀 How to Run
Clone the repository
Load the dataset in Python using Pandas
Run the Jupyter Notebook for cleaning and EDA
Execute SQL queries in PostgreSQL/MySQL/SQL Server
Open the Power BI (.pbix) file to explore the dashboard
